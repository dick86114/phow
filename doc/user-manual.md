# Phow 用户手册

## 1. 访问与登录
- **访问地址:** 浏览器打开部署地址（如 `http://localhost`）。
- **登录:** 点击右上角“登录”图标。
  - 初始管理员账号请参考部署文档或数据库初始化设置。
  - 登录失败会有明确提示。

## 2. 浏览照片
- **首页:**
  - **瀑布流/网格视图:** 点击右上角切换按钮可改变照片展示方式。
  - **热力图:** 顶部日历热力图展示每日上传活跃度。
    - **筛选:** 点击热力图上的特定日期，下方列表将只显示当天的照片。
    - **清除筛选:** 点击日期标签旁的“X”或再次点击日期可取消筛选。
- **详情页:**
  - 点击照片进入详情页。
  - 查看照片大图（支持原图下载）、EXIF 信息、AI 生成的故事。

## 3. 管理员功能
### 3.1 上传照片
1. 登录后点击顶部导航栏的“上传”按钮。
2. 拖拽或选择照片文件。
3. **AI 智能识别:**
   - 系统会自动尝试读取 EXIF。
   - 如果信息缺失，点击“AI 智能识别”按钮，系统将分析图片水印和内容，自动补充器材信息并生成一段情感故事。
4. 填写或确认信息后点击发布。

### 3.2 编辑与删除
- **编辑:** 在照片详情页，管理员可点击“编辑”按钮修改信息或重新进行 AI 识别。
- **删除:** 在详情页底部或编辑模式下，点击红色“删除”按钮即可永久删除照片。

## 4. 个性化设置
- **主题切换:** 点击顶部导航栏的太阳/月亮图标，可切换 浅色/深色/跟随系统 模式。
